# This function generates the enhancedVolcanoplot

This function generates the enhancedVolcanoplot

## Usage

``` r
plot_Volcano(table, feature_name, pvalcutoff = 0.05, x = "lodds", y = "p")
```

## Arguments

- table:

  Table with the test Results

- feature_name:

  column with names

- pvalcutoff:

  p-value threshold

- x:

  x-axis column name

- y:

  y-axis column name

## Value

R default plot
