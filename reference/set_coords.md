# This function assigns coordinates in the sankey plots for the genes on the dataframe

This function assigns coordinates in the sankey plots for the genes on
the dataframe

## Usage

``` r
set_coords(df, type)
```

## Arguments

- df:

  dataframe with genes contained in the sankey plot

- type:

  string defining the type of the genes inside the dataframe (L, R or
  TF)
